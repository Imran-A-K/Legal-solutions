Difference between FlexBox and Grid : Dimensions define the primary difference between Flexbox and CSS Grid. Flexbox are designed specifically for one-dimensional layouts, while CSS Grid is engineered to enable two-dimensional layouts.

Media is used for making the site responsive. It gives you a way to apply CSS only when the browser and device environment matches a rule where you specificly want it to apply.

CSS Box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used to create the design and layout of web pages. The box is composed of four parts (or areas), defined by their respective edges: the content edge, padding edge, border edge, and margin edge. This can be used to find and view the properties of a particular element.

Semantic tag are the elements in Html that have meaningful names which tells about type of content. For example header, footer, table, … etc. HTML5 introduces many semantic elements such as article aside,details,figcaption etc which make the code easier to write and understand for the developer as well as instructs the browser on how to treat them.
